# 插件 api

| 命令          | 功能                   |     |
| ------------- | ---------------------- | --- |
| request.offer | 请求服务端创建 offer。 |     |
| play.file     | 播放指定文件。         |     |
| pause.file    | 暂停播放。             |     |
| resume.file   | 恢复播放。             |     |
| stop.file     | 停止播放。             |     |

播放文件时会启动一个播放线程，结束或停止播放时释放线程。

## play.file

指定文件参数。

# 事件

|               |                    |     |
| ------------- | ------------------ | --- |
| launch.ffmpeg | 已经启动播放线程。 |     |
| exit.ffmpeg   | 已经启动播放线程。 |     |
